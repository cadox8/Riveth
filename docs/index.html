Requirements

You will need the following things to use the plugin:

  1. Spigot 1.12.2 (Should work on 1.9 - 1.12.2)
  2. Java 8
  3. MySQL
  4. Spigot/PaperSpigot

This docs are still in development, sorry for outdated information, I will put it ASAP.

Getting Started #back to top

For server owners

First, you will need to download the plugin from Spigot. Put it into your plugins folder and restart the server. Once all files has been created, you can configure them. When you finish, enable your server and you're done!
