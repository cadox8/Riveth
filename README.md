# Riveth
Minecraft server command mod.

### Dependencies
To run this software you will need:

- Spigot 1.12.2 (Should work on 1.9 - 1.12.2)
- Java 8
- MySQL

### Build
To build with Maven, use the command
```sh
$ mvn clean install
```
